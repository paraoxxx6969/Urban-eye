
# 🏙️ Urban Eye — Smart City Platform

> **Your city. Your voice. Your data.**

Urban Eye is a citizen-first governance platform that empowers communities to report civic issues, track real-time progress, and shape their neighbourhood's future — together. Built with a modern, sleek dark-themed UI.

## ✨ Features

- **Issue Reporting** — Report civic issues with location tagging, categories (Traffic, Safety, Environment), and priority levels
- **Interactive Dashboard** — Real-time stats on active reports, resolved issues, citizen trust score, and community points
- **Kanban Board** — Drag-and-drop issue tracking across New, In Progress, and Resolved columns
- **City Map** — Geospatial visualization of reported issues across the city
- **Rewards & Gamification** — Earn civic points, unlock achievement badges, and climb rank tiers from Newcomer to City Guardian
- **User Profiles** — Personalized profiles with contribution history
- **Social Auth** — Sign in with Google or GitHub

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/paraoxxx6969/Urban-eye.git
cd Urban-eye

# Install dependencies
npm i

# Start the development server
npm run dev
```

## 🛠️ Tech Stack

- **Frontend** — React + TypeScript
- **Build Tool** — Vite
- **Styling** — CSS with PostCSS
- **Deployment** — Netlify
